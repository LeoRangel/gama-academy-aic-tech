# Projeto ICarros FULL JS

> Aplicação javascript desenvolvida com objetivo de estudar a configuração de webpack e babel

## Linguagens e tecnologias

- JavaScript
- Sass
- Webpack
- Babel
- Axios

## Como executar a aplicação

#### Para buildar usando o Webpack:
`npm run build`

#### Para testar o build:
`cd dist && serve -s`

#### Para rodar o Sass:
`npm run sass`

#### Para rodar o app:
`npm start`
